Installation
=====

To install pflacco, first create a new virtual environment via pip or conda, activate and install it.
For Windows:
.. code-block:: console

   python -m venv venv

   "venv/Scripts/activate"

   python -m pip install pflacco

For Mac/Linux:
.. code-block:: console

   python3 -m venv venv

   source venv/bin/activate

   python -m pip install pflacco
